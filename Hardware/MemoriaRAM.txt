Memória RAM (Random Access Memory)
A Memória RAM (Random Access Memory) é um tipo de memória volátil e temporária utilizada pelo computador para armazenar dados e instruções 
que estão sendo processados ativamente. É diferente do armazenamento permanente, como o disco rígido (HD) ou o SSD, porque os dados na RAM
são acessados muito mais rapidamente, mas são perdidos quando o computador é desligado ou reiniciado.

#Nomeclatura da Memória RAM
Ex: Memória XPG Gammix D30, 8GB, 2666Mhz, DDR4, CL16
    
    - Memória XPG Gammix D30 --> É o nome da Memória
    - 8GB       --> É a quantidade de armazenamento
    - 2666Mhz   --> É o Clock / Frequência de operação (2666Mhz ou 2.66Ghz)
    - DDR4      --> É a Tecnologia de Transmissão de Dados
    - CL16      --> É a Latência
