!Teclas de Atalhos do Windows

Ctrl + A --> Seleciona tudo
Ctrl + C --> Copia
Ctrl + D --> Deleta
Ctrl + P --> Imprime
Ctrl + S --> Salva um arquivo
Ctrl + V --> Colar
Ctrl + X --> Recorta e cola
Ctrl + Y --> Refaz uma ação
Ctrl + Z --> Desfaz uma ação
Ctrl + Shift + Alt + Windows + L  --> Abre a página do Linkedin

Windows + Qualquer seta --> Move a janela pra qualquer direção da tela
Windows + A --> Abre as notificações do lado direito
Windows + B --> Usa a barra de tarefas(a parte de baixo do notebook)
Windows + D --> Mostra a área de trabalho
Windows + E --> Abre o Explorador de Arquivos
Windows + G --> Faz a gravação da tela da máquina
Windows + R --> Executar e digitar qualquer programa
Windows + I --> Abre as configurações gerais do sistema
Windows + L --> Abre a tela de login da máquina
Windows + P --> Mostra os tipos de tela
Windows + T --> Escolhe qualquer janela aberta
Windows + . --> Abre a caixa de Emojis

Shift + Del --> Deleta Permanentemente
Alt + P  --> Mostra o que tem no arquivo
Alt + F4 --> Deleta qualquer janela aberta
Ctrl + Shift + N --> Cria pasta
